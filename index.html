iTweaksUninstaller

Hello there! I'm iTweaksUninstaller, the open source application made for Windows users, by Leonardo Alfreducci, to simple uninstall all tweaks in once time on your jailbroken iOS device.

I recommend to install safestart and *NO PLS RECOVERY * (optional) from angelXwind Repository, because if your device still in boot loop, you can simple remove all tweaks on it with this application. After you can use Putty to run command killall sshd command.
